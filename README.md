# Interactive Tech Blog

このブログについてはこちら。

# 開発

Node.jsが必要です。バージョンはpackage.jsonのvoltaの項目で確認できます。

```
$ npm i
$ npm run dev
```

Voltaは[こちら](https://volta.sh/)でインストールできます。

# ビルド

```
$ npm run build
$ npx serve dist
```
